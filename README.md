# Computer Folder Service（电脑文件夹服务）

## 这是干什么的？

这是装在 **苹果 Mac 电脑** 上的一个小程序。它的作用可以概括成一句话：

> **让你在手机上（或别的设备上）能访问你 Mac 里「你亲自勾选过」的照片，其它照片别人拿不到。**

打个比方：Mac 像一间屋子，照片在柜子里。这个程序不会把整间屋子敞开，而是你只挑出允许给别人看的那几张，然后临时开一扇「小窗」；只有连上这扇窗的设备，才能按你的名单取走这些图。

目前主要支持 **图片**（如 jpg、png、webp、heic 等）。以后可能会扩展更多文件类型或视频。

---

## 适合什么场景？

- 手机 App 需要从 Mac 拉取相册里的部分图片（例如和本仓库配套的客户端）。
- 不想把整个文件夹都暴露到网上，只想 **精确控制** 哪些图可以被访问。
- Mac 和手机（或其它设备）在 **同一个局域网**（通常就是连同一个 WiFi）。

---

## 怎么用？（按顺序做就行）

1. **在 Mac 上打开本程序**  
   窗口标题一般为「Computer Folder Service」。

2. **选一个「根目录」**  
   点 **「目录」** 按钮，选你放照片的那个文件夹（建议用按钮选，不要手打路径，避免打错）。  
   之后你能勾选的照片，都必须在这个文件夹 **里面**（或子文件夹里）。

3. **在下方网格里浏览、勾选照片**  
   有子文件夹时会出现一排按钮，点进去再选图。勾上的图会出现在 **「已选」** 那一行；不要的可以在已选区域里删掉。

4. **确认网络和地址**  
   界面上会显示本机在内网里的地址（IP）。**一般不用改 IP**，保持默认即可。  
   若提示端口被占用，再改 **端口** 数字即可。

5. **点「启动」**  
   启动后，同一 WiFi 下的客户端（例如手机 App）用界面里显示的 **IP + 端口** 去连接，就能拿到你勾选的那份图片列表和原图。  
   **不用的时候点「停止」**，别人就无法再访问。

**重要：** Mac 防火墙若拦截入站连接，可能需要在「系统设置 → 网络/隐私与安全性 → 防火墙」里允许本程序，否则手机连不上。

## 界面各区域说明（对照窗口看）

| 区域 | 作用 |
|------|------|
| **IP / 端口** | 手机或别的设备要连的地址；IP 通常默认即可，端口冲突时再改。 |
| **启动 / 停止** | 打开或关闭「小窗」；停止后外部无法再访问。 |
| **根目录** | 允许浏览和勾选照片的最大范围；建议用「目录」按钮选择。 |
| **已选** | 最终允许对外提供的图片清单；可在这里预览小图或移除某张。 |
| **子目录按钮** | 在根目录下进入子文件夹。 |
| **图片网格** | 当前文件夹里的图片缩略图；勾选后进入「已选」。 |

说明：界面上是小缩略图，方便挑选；真正传给客户端的 **原图** 仍通过程序提供的地址按文件名获取（与以前客户端用法兼容）。

## 常见问题

- **手机连不上？**  
  先确认 Mac 和手机在同一 WiFi；服务已点「启动」；防火墙未拦截；IP/端口与客户端填写一致。

- **端口填什么？**  
  默认即可。若启动失败或提示占用，换一个没被占用的数字（例如 8081、9090），客户端里也要改成同一个端口。

- **能访问我整台电脑的文件吗？**  
  不能。对外能列出的「候选图」来自根目录下的扫描结果，而真正开放下载的，只有你 **勾选进「已选」** 的那部分。

## 给开发者看的（可选）

- **运行 / 打包（Gradle）**  
  在项目目录执行 `./gradlew run` 可本地运行；打包 macOS 安装包可参考 Compose Desktop 的 `packageDmg` 等任务。

- **HTTP 接口（摘要）**  
  `GET /health` 健康检查；`GET /selected` 当前勾选图片的 JSON 列表（含下载 URL）；`GET /images?offset=` 分页列出根目录下所有图片；`GET /image?name=相对路径` 返回磁盘上的原图文件。

- **仓库关系**  
  README 中曾提及可作为某示例客户端（如 MvvmSample）在 Mac 端对应的服务；客户端需自行配置为上述地址与端口。
